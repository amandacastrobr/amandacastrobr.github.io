MOLDELANDO EN FUSION

Archivo para imprimir moldes

El primer molde nos sirve para mostrar la el top y bottom layer, es importante darles grosor para darle estabilidad a la figura, tambíen giré la figura en 180 grados para la impresion.

Acá se logra apreciar el infill, en este aso usamos 10% y le dimos forma triangular

Finalmente, en este caso se ve el uso de soportes para lograr que las figuras que no tienen caras planas, o que tengas relieves hacia los lados, puedan ser impresas de igual manera
